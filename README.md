# TASM
# Sample Cases and MDAS using TASM
# How to execute tasm compiler in (cmd) 
# Providing Hills.bat file 
  Make sure tasm folder is already inside local disc C.
  C:\TASM\
  
# Step 1.
  cd -ENTER
  
  cd TASM -ENTER
  
  edit –ENTER

# Step 2.
  TO RUN .asm
  
  hills case -enter
  case.exe -enter

# Step 3.
  TO EDIT
  
  edit case.asm –enter
